# razerclient.github.io

Ez a mappa a GitHub Pages-re szant RazerClient weboldal.

## Javasolt repo nev

`razerclient.github.io`

## Feltoltes

1. Hozz letre egy publikus GitHub repositoryt `razerclient.github.io` neven.
2. Masold ebbe a repositoryba ennek a mappanak a tartalmat:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.nojekyll`
3. Pushold fel a `main` branchre.
4. A GitHub Pages automatikusan ki fogja adni a site-ot.

## Varhato cim

`https://razerclient.github.io`

## Megjegyzes

A `CNAME` fajlt most szandekosan nem hasznaljuk, mert elso korben nem sajat domainre, hanem `github.io`-ra megy az oldal.
